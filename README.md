# 🚀 TRANSIFY – AI-Powered Transition Engine

> **One platform. Every life transition.**

TRANSIFY is an AI-powered Transition Engine that helps people navigate major life transitions with confidence. Whether someone is relocating for work, starting higher education, changing careers, or entering a new phase of life, TRANSIFY creates a personalized roadmap by coordinating multiple services through intelligent AI agents.

Instead of searching across different apps for jobs, housing, finances, healthcare, transportation, and skill development, users receive a unified transition plan tailored to their needs.

---

## 🌍 Problem Statement

Life transitions are complex and stressful because existing digital platforms solve only isolated problems. People often switch between multiple applications to find employment, accommodation, financial guidance, transportation, and other essential services. This fragmented experience results in confusion, delays, poor decision-making, and missed opportunities.

TRANSIFY solves this challenge by bringing everything together into one intelligent platform.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/transify.git
cd transify
```

### Backend

```bash
cd backend

python -m venv .venv

source .venv/bin/activate
# Windows
# .venv\Scripts\activate

pip install -r requirements.txt

uvicorn main:app --reload
```

Backend runs at:

```text
http://127.0.0.1:8000
```

---

### Frontend

```bash
cd frontend

npm install

npm run dev
```

Frontend runs at:

```text
http://localhost:3000
```
